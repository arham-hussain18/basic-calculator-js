# 🧮 Responsive Calculator using HTML, CSS & JavaScript

A modern and responsive Calculator built using **HTML5**, **CSS3**, and **JavaScript**. This project performs basic arithmetic operations with a clean user interface, real-time display, keyboard support, and responsive design for all screen sizes.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🟰 Real-Time Result Display
- ⌨️ Keyboard Support
- 🧹 Clear (AC) Function
- ⌫ Delete/Backspace Function
- 📱 Fully Responsive Design
- 🎨 Modern User Interface
- ✨ Smooth Button Animations
- ⚡ Fast and Lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/basic-calculator-js.git
```

### Navigate to the Project

```bash
cd basic-calculator-js
```

### Run the Project

Open the `index.html` file in your web browser.

No installation or external libraries are required.

## 🎯 Supported Operations

The calculator supports:

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Decimal Numbers
- Percentage (Optional)
- Clear Screen (AC)
- Delete Last Character
- Keyboard Input

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|------|--------|
| 0-9 | Numbers |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| Enter | Calculate Result |
| Backspace | Delete Last Character |
| Escape | Clear Screen |
| . | Decimal Point |
